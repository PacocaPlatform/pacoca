Cole o código do mapa em formato Grid ASCII (incluindo cabeçalhos) ou JSON estruturado para importá-lo no editor.

Como compilar este nível para o Godot:

  1. Salve/Baixe o arquivo de texto como level_04_map.txt.
  2. Coloque o arquivo na pasta do projeto em:
    src/scripts/levels/.
  3. Abra seu terminal na raiz do projeto e execute a compilação:
python scripts/convert_map.py --input scripts/levels/level_04_map.txt --level 04
Métricas Importantes (Platformer Kit)
  • Grade X (Largura): Cada bloco # ou rampa / mede 2m.
  • Grade Y (Altura): Cada linha mede 1m.
  • Salto Médio: 4m horizontalmente (parado), até 15m (com velocidade máxima).
  • Molas: Verticais lançam a 22m de altura. Diagonais para cruzamento rápido.
  • Queda Fatal: Evite plataformas em Y < -15m.
