Métricas importantes (Platformer Kit)
  • Grade X (largura): cada bloco # ou rampa / mede 2m.
  • Grade Y (altura): cada linha mede 3m (ystep: 3.0).
  • Pulo médio: 4m na horizontal (parado), até 15m (em velocidade máxima).
  • Molas: verticais lançam até 22m; diagonais para travessias rápidas.
  • Queda fatal: evite plataformas em Y < -15m.
